# Language: English

Use concise IEEE English:

- Prefer active technical verbs: formulate, derive, estimate, allocate, optimize, align, suppress, quantize, calibrate.
- Use present tense for paper structure and method facts; past tense for completed experiments when needed.
- Keep noun stacks short when they obscure meaning.
- Define each acronym at first use in the abstract and again in the body if the abstract is standalone.
- Use "we" sparingly for actions performed in the paper, not for hype.
