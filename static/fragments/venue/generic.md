# Venue: Generic IEEE

Use IEEE journal style:

- Abstract: problem, gap, method, key evidence, boundary.
- Introduction: move from wireless setting to technical bottleneck, not broad slogans.
- Contributions: two to four bullets, each with a technical delta and evidence hook.
- Related work: group by limitation or method family.
- Experiments: compare against direct competitors and explain fairness.
- Conclusion: restate bounded contribution and main evidence; avoid future-work padding.
