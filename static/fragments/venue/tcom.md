# Venue: IEEE Transactions on Communications

TCOM-style writing should foreground communication-theoretic clarity.

Emphasize:

- Signal model, estimation/detection/decoding formulation, or information-theoretic framing.
- Analytical derivation, convergence, complexity, or performance bounds when available.
- Clean notation and rigorous assumptions.

For learning-based work, explain why the learning component is necessary for the communication problem.
