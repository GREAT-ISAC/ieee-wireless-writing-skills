# Venue: IEEE Transactions on Wireless Communications

TWC-style drafts need depth in wireless modeling, algorithmic novelty, and experiments.

Emphasize:

- Clear system and channel model.
- Direct same-task wireless baselines.
- Mathematical or algorithmic contribution beyond implementation engineering.
- Sensitivity to SNR, mobility, antennas, pilots, blockage, or interference when relevant.
- Reproducible simulation details.

Weak signals:

- Only comparing to generic ML baselines.
- No discussion of CSI acquisition, pilot overhead, or channel assumptions.
- Claims of generality from one channel model or one dataset.
