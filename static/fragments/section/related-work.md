# Section: Related Work

Synthesize by technical limitation rather than listing papers.

Common grouping patterns:

- Model-based signal processing or optimization.
- Learning-based estimators or controllers.
- Domain adaptation/generalization methods.
- Hardware-aware or practical-overhead methods.
- Direct same-task competitors.

Each paragraph should end with the gap your paper addresses. Avoid attacking strawman baselines; state what prior work does well before naming the limitation.
