# Section: Experiments

Results prose should answer reviewer questions, not narrate plots chronologically.

Minimum content:

- Setup: channel/data, SNR, antenna count, bandwidth, mobility, training split, and implementation details.
- Baselines and fairness.
- Metrics.
- Main comparison.
- Ablation.
- Sensitivity or robustness.
- Complexity/latency if claimed.

State exact conditions for each gain. Do not generalize beyond the tested range.
