# Section: System Model

The system model must let reviewers reproduce the assumptions.

Include as relevant:

- Nodes, antennas, RF chains, users, RIS elements, sensors, subcarriers, time slots, and bandwidth.
- Channel model and notation.
- Signal model with noise/interference.
- CSI availability and pilot/feedback mechanism.
- Optimization or learning objective.
- Constraints and known/unknown variables.

Use notation consistently. Do not introduce a variable unless it appears in equations, algorithm, or experiments.
