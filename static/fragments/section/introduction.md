# Section: Introduction

Build the introduction as a technical funnel:

1. Wireless system context.
2. Bottleneck that matters under that context.
3. Prior approach groups and their limitations.
4. Specific unresolved gap.
5. Proposed method and why it addresses the gap.
6. Contributions and evidence.

Contribution bullets must be auditable: each bullet needs a method/theory/system delta and an evidence hook.
