# Area: mmWave and THz

mmWave/THz work must surface propagation and hardware constraints.

Check:

- Blockage, beam squint, beam training overhead, phase noise, molecular absorption, path sparsity, or near-field effects.
- Hybrid beamforming or array constraints.
- Mobility and beam tracking if practical deployment is claimed.
- Dataset or ray-tracing setup when using learned methods.

Avoid using generic sub-6 GHz assumptions for mmWave/THz claims.
