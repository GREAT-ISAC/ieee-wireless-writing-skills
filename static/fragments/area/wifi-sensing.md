# Area: WiFi Sensing

WiFi sensing papers must define the sensing task and domain boundary.

Specify:

- Device, band, bandwidth, antenna/subcarrier count, sampling rate, and CSI extraction tool if known.
- Task: gesture, activity, respiration, localization, identification, tracking, or environment sensing.
- Domain split: cross-user, cross-location, cross-device, cross-day, line-of-sight/non-line-of-sight.
- Metrics: accuracy, F1, localization error, respiration error, latency, or robustness.

Do not claim generalization from random splits that leak user, location, or session identity.
