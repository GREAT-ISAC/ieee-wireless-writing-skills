# Area: Radio ML

Radio ML work must justify the learning component in wireless terms.

Require:

- Input representation and preprocessing.
- Training/test split that matches the claimed deployment.
- Model size, latency, and inference cost if deployment is claimed.
- Baselines: classical signal processing, optimization, compact ML, and recent neural methods as relevant.
- Ablations that isolate architecture, loss, augmentation, and domain adaptation contributions.

Avoid "black-box improvement" prose. Explain what channel structure, invariance, or prior the model exploits.
