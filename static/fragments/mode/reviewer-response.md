# Mode: Reviewer Response

Use a respectful, evidence-bound point-by-point response.

Workflow:

1. Extract editor instructions and reviewer comments separately.
2. Assign IDs: `E.1`, `R1.1`, `R1.2`, `R2.1`, etc.
3. Classify each item: novelty, baseline, experiment, theory, clarity, notation, citation, figure, grammar, or scope.
4. Draft a response with: acknowledgement, action taken, manuscript location, and evidence.
5. If an experiment or citation is not actually done, do not claim it. Mark `AUTHOR_INPUT_NEEDED`.
6. Keep disagreement narrow and technical.

Load [references/reviewer-response.md](../../../references/reviewer-response.md) before finalizing.
