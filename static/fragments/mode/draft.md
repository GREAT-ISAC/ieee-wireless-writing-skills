# Mode: Draft

Before writing prose, create a one-sentence argument:

`In [wireless setting], prior methods fail because [specific limitation]; this paper proposes [technical delta], supported by [evidence], under [boundary].`

Draft sections in this order unless the user requests otherwise:

1. Contribution bullets and evidence hooks.
2. System model and problem formulation.
3. Method.
4. Experiments.
5. Introduction and related work.
6. Abstract, title, and conclusion.

Use paragraph structure:

- Topic sentence with the local claim.
- Technical mechanism or evidence.
- Boundary, contrast, or transition.

Avoid empty phrases: "to the best of our knowledge", "significant improvement", "massive potential", "novel framework", and "comprehensive experiments" unless made specific.
