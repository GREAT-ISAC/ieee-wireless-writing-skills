# Mode: Audit

Audit as an IEEE reviewer would:

- Novelty: Is the delta beyond application relabeling, hyperparameter tuning, or combining known modules?
- Evidence: Does every superiority claim point to a fair baseline, ablation, sensitivity test, or theoretical result?
- Assumptions: Are channel model, CSI availability, synchronization, mobility, pilot overhead, and hardware constraints explicit?
- Reproducibility: Are datasets, code settings, splits, SNR range, antenna counts, bandwidth, and metrics specified?
- Writing: Are contribution bullets concrete, related work synthesized, notation consistent, and figures readable?

Return severe issues first. For each issue, provide a fix that can be applied to the manuscript.
