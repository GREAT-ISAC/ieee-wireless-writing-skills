# Claim-Evidence Audit

Use this matrix for contribution bullets, abstract claims, introduction claims, and reviewer-sensitive statements.

| Claim | Evidence type | Evidence location | Boundary | Citation role | Action |
| --- | --- | --- | --- | --- | --- |
|  | theorem / algorithm / simulation / measurement / dataset / citation | Fig. / Table / Eq. / Sec. | channel, SNR, dataset, topology, device, or assumption | competitor / foundation / dataset / metric | keep / weaken / cite / test / remove |

Severity rules:

- Severe: superiority, novelty, practicality, or robustness claim has no direct evidence.
- High: evidence exists but uses unfair baseline, unclear metric, or hidden assumption.
- Medium: claim is true but too broad for tested conditions.
- Low: wording, citation placement, or notation issue.

Contribution bullet template:

`We propose [technical delta] for [wireless setting], which addresses [prior limitation]. Evidence: [theorem/algorithm/figure/table]. Boundary: [assumptions].`

Do not accept contribution bullets that only say "a novel framework", "extensive simulations", or "better performance".
